# 🌟 Systems Design Repository

Welcome to the Systems Design repository! This project explores and demonstrates
various systems design patterns and architectures through practical examples.
Each implementation showcases how different architectural decisions impact
system behavior, scalability, and maintainability.

## 🧠 Repository Overview

This repository contains hands-on examples of systems design principles:

1. **TinyURL (Monolithic Architecture)**: A URL shortening service demonstrating
   basic systems design concepts
2. **[Future additions]**: More examples of distributed systems, microservices,
   and beyond

## 🧪 Learning Objectives

By studying and implementing these examples, you will gain insights into:

- Trade-offs between different architectural approaches
- Designing scalable and resilient systems
- Service communication patterns
- Deployment strategies for various architectures

## 📖 Repository Structure

The repository contains the following key directories:

1. **tiny-url**: Implements TinyURL as a monolithic service
   - Demonstrates core systems design principles
   - Ideal for beginners learning systems design

## 🧠 Why Study Systems Design?

Understanding systems design is crucial for any developer working on complex
applications. This repository provides:

- Practical examples of theoretical concepts
- Clear demonstration of architectural trade-offs
- A foundation for exploring more advanced patterns

## 🔍 Explore the TinyURL System

The TinyURL system is an excellent starting point for understanding systems
design because:

- It's a simple problem with multiple solution paths
- It demonstrates core concepts applicable to more complex systems
- It allows you to see how architectural decisions impact system behavior

## 🧭 Getting Started

### TinyURL (Monolithic)

```bash
# Clone the repository
🦄❯ git clone https://github.com/verofa/systems-design.git

# Change directory
🦄❯ cd systems-design/tiny-url/monolithic

# Start the service
🦄❯ docker build -t tinyurl-monolithic .
🦄❯ docker run -p 5000:5000 tinyurl-monolithic

```
